# Perceptron (Epoch-Based Learning)

## Aim

To implement the Perceptron algorithm using epoch-based updates.

## Description

This implementation demonstrates a basic perceptron model trained using multiple passes (epochs) over the dataset. The model updates its weights based on prediction errors to learn a linear decision boundary.

## Working

* Initialize weights and bias
* Loop through dataset for multiple epochs
* Compute weighted sum
* Apply step activation function
* Update weights if prediction is incorrect
