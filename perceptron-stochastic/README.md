# Perceptron (Stochastic Learning)

## Aim

To implement the Perceptron algorithm using stochastic (random sample) updates.

## Description

This implementation uses stochastic learning where the model updates weights using one randomly selected data point at a time instead of the full dataset.

## Working

* Initialize weights and bias
* Randomly select a data point
* Compute weighted sum
* Apply step activation function
* Update weights immediately if prediction is incorrect
